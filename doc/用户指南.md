## 1. 系统简介

Memo 是一款基于 Web 的待办任务管理系统，支持多用户协作，提供任务创建、编辑、删除、状态管理、筛选排序、附件上传等功能。

### 主要功能

- **任务管理**: 创建、编辑、删除任务，支持优先级和截止时间
- **状态流转**: 待办 → 进行中 → 已完成/已取消
- **附件上传**: 支持图片、视频、文档等多种文件格式
- **筛选搜索**: 按状态、优先级筛选，支持关键词搜索
- **拖拽排序**: 灵活调整任务顺序
- **统计面板**: 直观查看任务统计信息
- **多用户支持**: 用户之间数据完全隔离
- **管理员功能**: 用户管理、系统配置、系统统计

### 技术栈

| 层级 | 技术 |
|------|------|
| 前端 | Vue 3 + Element Plus |
| 后端 | Python FastAPI |
| 数据库 | SQLite |
| 认证 | JWT |

## 3. 用户管理

### 3.1 注册新账户

1. 点击登录页面的「没有账号？去注册」
2. 填写注册信息：
   - **用户名**: 4-20位，支持字母、数字、下划线
   - **密码**: 8-32位，必须包含字母和数字
   - **确认密码**: 再次输入密码
   - **邮箱**: 选填，用于密码找回
   - **昵称**: 选填，页面显示名称
3. 点击「注册」完成注册
4. 注册成功后自动跳转到登录页面

### 3.2 登录

1. 在登录页面输入用户名和密码
2. 可勾选「记住我」延长会话有效期
3. 点击「登录」进入系统

**登录安全规则**:
- 连续 5 次登录失败，账户将被锁定 15 分钟
- 错误提示不透露具体是用户名错误还是密码错误

### 3.3 修改密码

1. 点击顶部导航栏右侧的用户名
2. 选择「个人设置」
3. 在修改密码区域：
   - 输入旧密码
   - 输入新密码（8-32位）
   - 确认新密码
4. 点击「保存」

**首次登录强制改密**: 管理员初始密码或被重置密码后，首次登录必须修改密码。

### 3.4 修改昵称

1. 进入「个人设置」页面
2. 在账号信息区域，点击昵称旁边的「编辑」按钮
3. 输入新昵称
4. 点击「保存」

### 3.5 退出登录

1. 点击顶部导航栏右侧的用户名
2. 选择「退出登录」

## 4. 任务管理

### 4.1 创建任务

1. 点击任务列表上方的「新建任务」按钮
2. 在弹出的对话框中填写任务信息：
   - **标题**: 必填，最大100字符
   - **描述**: 选填，最大2000字符
   - **优先级**: 默认「中」，可选择「高」「中」「低」
   - **截止时间**: 选填，设置日期和时间
3. 点击「创建」完成

**任务默认状态**: 新建任务默认为「待办」状态。

### 4.2 查看任务

在任务列表中查看所有任务：
- **标题**: 任务名称
- **描述**: 鼠标悬停可查看完整描述
- **优先级标签**: 高=红色，中=橙色，低=绿色
- **状态标签**: 显示当前状态
- **截止时间**: 过期任务红色高亮显示
- **附件图标**: 有附件时显示📎及数量

### 4.3 编辑任务

1. 点击任务行的编辑按钮（铅笔图标）
2. 在弹出的对话框中修改任务信息
3. 点击「保存」

**可编辑字段**: 标题、描述、优先级、截止时间

### 4.4 删除任务

1. 点击任务行的删除按钮（垃圾桶图标）
2. 在确认对话框中点击「确定」
3. 任务将被软删除（数据保留，前端不再显示）

### 4.5 状态切换

任务支持以下状态流转：

```
待办 → 进行中（开始处理）
待办 → 已取消（放弃任务）
进行中 → 已完成（任务完成）
进行中 → 待办（回退）
已取消 → 待办（恢复任务）
已完成 → 无（终态，不可更改）
```

**操作方式**:
1. 点击任务的状态标签
2. 在下拉菜单中选择目标状态
3. 状态自动更新

### 4.6 筛选任务

**按状态筛选**:
- 点击统计卡片可快速筛选对应状态
- 例如：点击「待办」卡片，只显示待办任务
- 点击「全部」卡片，显示所有任务

**按优先级筛选**:
- 在筛选工具栏选择优先级下拉框
- 选择「高」「中」「低」或「全部」

**关键词搜索**:
- 在搜索框输入关键词
- 实时搜索任务标题和描述

### 4.7 拖拽排序

1. 鼠标拖动任务左侧的拖拽手柄（☰图标）
2. 拖动到目标位置
3. 松开鼠标后自动保存新顺序
4. 刷新页面后排序保持不变

### 4.8 附件管理

**上传附件**:
1. 编辑任务，进入编辑对话框
2. 点击「上传附件」按钮
3. 选择文件或拖拽文件到上传区域
4. 支持格式：图片(JPG/PNG/GIF等)、视频(MP4/AVI等)、文档(PDF/DOC等)

**文件大小限制**:
| 类型 | 默认限制 |
|------|---------|
| 图片 | 10 MB |
| 视频 | 200 MB |
| 其他 | 50 MB |

**删除附件**:
1. 在附件列表中点击删除按钮
2. 确认后附件将被删除

## 5. 统计面板

### 5.1 统计卡片说明

任务列表页面顶部显示统计卡片：

| 卡片 | 说明 | 计算方式 |
|------|------|---------|
| 全部 | 任务总数 | 所有未删除任务 |
| 待办 | 待办任务数 | 状态=待办 |
| 进行中 | 进行中任务数 | 状态=进行中 |
| 已完成 | 已完成任务数 | 状态=已完成 |
| 已过期 | 已过期任务数 | 截止时间已过且未完成 |

### 5.2 快速筛选

点击任意统计卡片，可快速筛选该类型任务：

```
点击「待办」卡片 → 只显示待办任务
点击「已过期」卡片 → 只显示已过期任务
点击「全部」卡片 → 显示所有任务
```

## 6. 管理员功能

> 只有管理员用户才能访问以下功能

### 6.1 用户管理

入口：「管理后台」→「用户管理」或点击任务列表中的「管理后台」按钮

**查看用户列表**:
显示所有注册用户信息：
- 用户名、昵称、邮箱
- 角色（管理员/用户）
- 状态（正常/已禁用）
- 注册时间

**禁用/启用用户**:
1. 点击目标用户行的「禁用」按钮
2. 确认操作
3. 被禁用的用户将无法登录

**重置用户密码**:
1. 点击目标用户行的「重置密码」按钮
2. 输入新密码（8-32位）
3. 确认后密码将被重置
4. 该用户首次登录时需要强制修改密码

### 6.2 系统配置

入口：「管理后台」→「系统配置」

**附件上传限制**:
| 配置项 | 默认值 | 可调范围 |
|--------|--------|---------|
| 图片最大文件大小 | 10 MB | 1-100 MB |
| 视频最大文件大小 | 200 MB | 10 MB - 2 GB |
| 其他附件大小 | 50 MB | 1-500 MB |
| 每任务最大附件数 | 10 个 | 1-50 个 |

**保存配置**:
1. 修改各项数值
2. 点击「保存配置」
3. 配置即时生效

### 6.3 系统统计

入口：「管理后台」→「系统统计」

查看系统级别的统计数据：
- **总用户数**: 所有注册用户数量
- **总任务数**: 所有用户的任务总数
- **完成任务率**: 已完成任务占比
- **任务状态分布**: 各状态任务数量和占比图表

## 7. 常见问题

### Q1: 忘记密码怎么办？

请联系系统管理员重置密码。

### Q2: 账户被锁定怎么办？

账户因连续5次登录失败被锁定15分钟后会自动解锁。也可联系管理员手动解锁。

### Q3: 为什么任务状态无法切换？

请检查是否符合状态流转规则。例如：已完成的任务不能直接变为进行中，需要先恢复为已取消或待办。

### Q4: 附件上传失败怎么办？

1. 检查文件大小是否超过限制
2. 检查文件格式是否支持
3. 检查网络连接是否正常

### Q5: 统计数据不准确怎么办？

统计数据会实时更新。请刷新页面或重新登录获取最新数据。

### Q6: 如何彻底删除任务？

系统使用软删除，数据实际保留。如需彻底删除，请联系数据库管理员操作。

### Q7: 管理员账户密码忘记怎么办？

可以通过数据库直接重置：
```sql
UPDATE users SET hashed_password='$2b$12$...' WHERE username='admin';
```
（需要先通过命令行生成新密码的哈希值）

### Q8: 支持移动端吗？

系统支持响应式布局，PC端体验最佳。移动端适配为低优先级。
